# Fashion MNIST Classification Project 

## Project Overview
This project focuses on building predictive models to classify images of clothing articles using the Fashion MNIST dataset (60,000 images). Developed as part of the "Laboratorio de Datos" course at Universidad de Buenos Aires (UBA).

## Key Results
- **Binary Classification (Trousers vs. T-shirts):** 96% Accuracy using KNN.
- **Multiclass Classification (10 classes):** 81% Global Accuracy using Decision Trees.

## Tools Used
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Scikit-learn (PCA, KNN, DecisionTree).
- **Techniques:** EDA, Data Wrangling, Hyperparameter Tuning (RandomizedSearchCV).

## Files
- `Fashion_MNIST_Analysis.ipynb`: Complete Jupyter Notebook with code and outputs.
- `informe_Fashion_MNIST.pdf`: Detailed technical report (Spanish) with visualizations and conclusions.
